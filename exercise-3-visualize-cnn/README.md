### Accuracies

| Learning rate  | Epochs   | Accuracy |
|:--------------:|:--------:|:--------:|
| 1e-6           | 1        | 6%       |
| 1e-3           | 20       | 97%      |


With higher number of epochs (i.e. 20), accuracy of the model increased to 97%. In contrary, when the model is trained for 1 epoch only, accuracy is very low (i.e. 6%).
